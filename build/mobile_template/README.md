# OpenReality Mobile Export

This directory is generated by `orcli build mobile`. It wraps the WebGPU/WASM
web build in a native WebView shell via Capacitor.js.

## Requirements

- Node.js 18+
- Android: Android Studio + SDK 24+
- iOS: Xcode 15+ (macOS only)

## Setup

1. Build the web target first:
   ```
   orcli build web <scene.jl> --release
   ```

2. Copy the web build output into `web/`:
   ```
   cp -r build/web/* web/
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Build for your target platform:
   ```
   # Android
   npm run build:android

   # iOS (macOS only)
   npm run build:ios
   ```

## Notes

- WebGPU requires Chrome 113+ on Android or Safari 18+ on iOS
- Performance will be lower than native desktop builds
- Touch input is mapped to mouse events by the WASM runtime
